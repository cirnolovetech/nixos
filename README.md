# ❄️ My NixOS config with Home Manager & Flakes ❄️

## Features 
| Category         | Component     | 
|-----------------|--------------|
| **Window Manager** | Hyprland |
| **Status Bar** | Waybar |
| **Styling** | Stylix |
| **Theme** | Custom Nord Theme |
| **Cursor** | DMZ-White |
| **Terminal** | Kitty |
| **Shell** | Zsh |
| **Notification** | Dunst |
| **Lock** | Hyprlock |
| **Idle** | Hypridle |
| **Display Manager** | SDDM |
| **App Launcher** | Rofi Wayland |
